CSS Hangi İhtiyaç Sebebiyle Çıktı?

Html geliştikçe daha iyi bir görünüm ihtiyacı ortaya çıktı. CSS ilk kez 1994 tarihinde CERN’de Tim Berners-Lee ile beraber çalışan Hakon Wium Lie tarafından önerilmiştir. Başka biçimlendirme dilleri önerileride çıkmış ve W3C kendisine gelen önerileri inceleyerek 1996 da CSS önerisini sunmuştur.

CSS Ne İşe Yarar?

Cascading Style Sheets (Basamaklı Stil Şablonları ya da Basamaklı Biçim Sayfaları, bilinen kısa adıyla CSS), HTML'e ek olarak metin ve format biçimlendirme alanında fazladan olanaklar sunan bir işaretleme dilidir.
